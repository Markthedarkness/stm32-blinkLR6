# stm32-blinkLR6
Лабораторна робота 6 Широтно-імпульсна модуляція для управління силовими 
елементами на основі апаратних таймерів
Обладнання: STM32F103С8T6, CubeIDE
Завдання:
Запрограмувати генерацію ШІМ на PD12 (TIM2) з частотою 1 кГц, керування шириною 
імпульсу кнопкою PA0.
